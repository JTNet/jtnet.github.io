JTNet 개발자 페이지

JTNet 개발자를 위한 페이지 입니다.

JTNet의 개발자와 JTNet을 이용하는 외부 개발자를 위한 페이지입니다. 특히, JTNet의 온라인결제 서비스인 tPay를 이용하여 서비스를 개발하시는 분들을 위해 쉬운 개발 환경과 많은 정보를 제공하려고 노력하고 있습니다.

JTNet은 오픈소스 개발자분들의 참여를 참여하고 적극 지원합니다.

  • 저희 서비스를 보다 쉽게 사용하기 위해 개발된 오픈소스가 있으면 알려주세요.
  • JTNet은 외부 개발자의 참여를 환영하고 지원합니다.

tPay SDK 및 메뉴얼

  • PHP SDK 공식 기술 지원 사이트를 통해 다운받으실 수 있습니다.
  • JSP SDK 공식 기술 지원 사이트를 통해 다운받으실 수 있습니다.
  • ASP SDK 공식 기술 지원 사이트를 통해 다운받으실 수 있습니다.
  • Ruby SDK : 준비중입니다.
  • Python SDK : 준비중입니다.
